## Quick Start Path

1. **Learn** — Read `01` for fundamentals, use Claude as a tutor
2. **Practice** — Paper trade on Manifold Markets (free) or Alpaca (stocks)
3. **Build** — Use `06` to start building tools with Claude's help
4. **Automate** — Follow `02` and `04` to build bots and AI systems
5. **Predict** — Dive into `03` for prediction market strategies

## Key Tools to Set Up First

- **TradingView** — Free charting and analysis (see doc 05)
- **Alpaca** — Free stock trading API with paper trading (see doc 02)
- **Manifold Markets** — Free play-money prediction markets (see doc 03)
- **Python + Jupyter** — For building everything (see doc 04)

## Important Disclaimers

- This is research material, not financial advice
- Always start with paper trading before risking real money
- Never invest money you cannot afford to lose
- Past performance does not guarantee future results
- AI tools are not crystal balls — they assist research, not predict the future
